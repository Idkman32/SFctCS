# SFctCS
StockFish 11 Compatible To Chess Sites. I am not responsible if this extension leads to Unfair Advantage.
This is a Browser extension. To use it download the ZIP, extract it in to a empty folder. Delete README.md after you READ it. Go the extensions tab on chrome (or any browser) and enable developer mode.
Press load packaged item. Choose the folder you extracted. And its now a real extension on your browser.
